# Installation

## Requirements

- Node.js 14.0.0 or newer
- npm 6.14.0 or newer

## Install

```bash
npm install
```

## Run

```bash
npm start
```

## Test

```bash
npm test
```

## Build

```bash
npm run build
```

## Run production build

```bash
npm run serve
```

```bash
npm run deploy
```
